\section{Deployment Diagram}

The deployment diagram illustrates the physical architecture of the system and how different components are distributed across hardware nodes \cite{booch2005uml,owler2010uml}.

It highlights:
\begin{itemize}
	\item Client devices (web or mobile users)
	\item Application server
	\item Database server
	\item Communication protocols between components
\end{itemize}unication protocols between components
