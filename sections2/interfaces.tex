\section{User Interfaces}

This section presents the main user interfaces organized by the four primary actors of the system: \textit{Farmer}, \textit{Buyer}, \textit{Transporter}, and \textit{Ministry}.

These interfaces are derived from the design prototypes introduced in Section~\hyperref[sec:mockups]{1.6 (Mockups)}, ensuring consistency between the conceptual design and the final implementation.

Each actor interface is illustrated through both web and mobile views.
